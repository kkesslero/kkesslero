## 🔐 About Me

- 🎓 Computer Science graduate from Concordia University, Montreal.
- 🛡️ Aspiring cybersecurity professional, currently working as a NOC Operator at Premiere Networks (iHeartMedia).
- 🧪 Building a hands-on [cybersecurity home lab](https://github.com/kkesslero/cybersecurity-home-lab) — simulating attacks, building detection tools, and documenting everything.
- 📡 Passionate about network security, threat intelligence, and incident response.
- 🔑 CompTIA Security+ (in progress).

## 🔨 What I'm Building

- **SIEM Log Analyzer** — Python tool that parses auth and web logs to detect brute-force attacks and scanning patterns
- **Network IDS** — Real-time intrusion detection using Scapy (port scans, SYN floods, SSH brute force, suspicious DNS)
- **Honeypot + Threat Intel** — Cowrie SSH honeypot paired with a correlator that enriches attacker IPs via AbuseIPDB
- **More projects in progress...**
